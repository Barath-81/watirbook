## Alt

### Elements

    <img>
    <input type="button">

### Gems

watir, watir-webdriver

### HTML

    <img alt="bock">
    <input alt="doppelbock" type="button">

### What

string, regular expression

### String

    browser.img(:alt => "bock")
    browser.button(:alt => "doppelbock")

### Regular Expression

    browser.img(:alt => /bock/)
    browser.button(:alt => /doppelbock/)
